# DesiVastra Store 🛍️

A modern Indian fashion e-commerce platform built with PHP/MySQL.

## Features
- 🛒 Full e-commerce frontend (shop, product, cart, checkout)
- 👤 Customer registration & login
- 📦 Orders & tracking
- 💳 Razorpay payment integration
- 📧 Email notifications (SMTP)
- 🎟️ Coupon/discount system
- ⭐ Product reviews
- 🔧 Complete admin panel
  - Dashboard with analytics
  - Product & category management
  - Order management
  - Customer management
  - Reports & charts
  - Settings (payment, email, shipping)
  - Activity log

## Tech Stack
- **Backend:** PHP 8+, PDO/MySQL
- **Frontend:** HTML5, CSS3, Vanilla JS
- **Payments:** Razorpay
- **Email:** PHPMailer (SMTP)
- **Charts:** Chart.js

## Setup

1. Upload all files to your web hosting `public_html`
2. Create a MySQL database
3. Visit `https://yourdomain.com/install` to run the installer
4. Follow on-screen steps to configure DB, admin credentials & site URL
5. Login at `https://yourdomain.com/admin-login`
